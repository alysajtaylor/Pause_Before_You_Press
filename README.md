# Pause Before You Press

## Inspiration
I felt inspired by the **emails** my **university** sends out to bring **awareness to scams**. The emails often pose as official or trusted parties, include attachments and bait students into clicking links. Instead of just send fake scam emails, it would be helpful to **define strategies** to **identify scams** and how to handle them safely. 

## What it does
This website focuses on equipping users with the skills to **identify** scams and to **practice** those skills in a **safe environment**.
The website identifies _two strategies_ : _Pause_ and _SLAM_ that help identify scams. Then you can practice identifying the scams in a mini quiz. The quiz intentionally does not provide a score, as I felt a score might discourage people from **learning**.

## How it is built
This website is built using HTML, CSS and vanilla JavaScript.
Additionally, Git/GitHub for version control and Vercel for deployment. 

## Challenges
It was difficult to narrow down the scope of what I wanted to achieve. The project went through several versions, one including a practice space for learning to prompt AI. With too many ideas and **time constraints**, the goal of the website became _unclear_. I decided to focus on a single aspect of **online safety**.
